# Level Up

## Goal
Interview Chao briefly, then surface the single best next skill/connection/automation to build —
so the brain keeps compounding.

## Steps
1. Read current context, connections and skills so suggestions fit the real stack.
2. Ask (one sentence answers are fine):
   - What did you do 3+ times this past week?
   - What felt manual, boring, or copy-paste?
   - What would a smart intern have handled that you did yourself because explaining it was slower?
   - If demand 10×'d on Monday, what breaks first?
   - What one automated lever would most grow the business?
3. From the answers, propose 1–3 candidates. For the top pick, sketch it as a skill using the
   6-step frame: **name/trigger · goal · step-by-step · reference files · rules · improvement loop.**
4. Offer to build it now (do the task together once, then turn it into a `SKILL.md`).

## Rules
- Prefer the *boring, deterministic* option when it suffices — it usually beats a full agent.
- Recommend one thing to start; don't overwhelm with a backlog.
- Log the chosen direction in `decisions/decision-log.md`.
